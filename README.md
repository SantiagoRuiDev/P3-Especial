# 💡 Programación 3 - Trabajo Especial

## 🚀 Necesario para un correcto funcionamiento
- Tener instalado [Java JDK 21](https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html).
- Agregar las dependencias de [Maven](https://mvnrepository.com/)

## ⚙️ Consejos para testear el proyecto
Hay un un archivo llamado Test dentro de la estructura principal, dentro de este archivo se puede encontrar un pequeño testeo de cada servicio y algoritmo requerido en el enunciado.
En la carpeta Resources se puede encontrar los archivos CSV requeridos para la carga de datos de prueba.

## 📖 Generación de Documentación (Javadoc)

El proyecto incluye comentarios Javadoc en el código para explicar la lógica de los servicios y algoritmos implementados. Para generar el manual de documentación en formato HTML, ejecuta el siguiente comando en la raíz del proyecto:

``` bash
mvn javadoc:javadoc
```
Una vez finalizado, abre el archivo index.html que se encuentra en la siguiente ruta: target/reports/apidocs/index.html
